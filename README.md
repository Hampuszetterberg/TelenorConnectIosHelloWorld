# TelenorConnectIosHelloWorld
Hello world example for Telenor Connect iOS SDK

## Install and run

1. `git clone git@github.com:telenordigital/TelenorConnectIosHelloWorld.git`
2. `open TelenorConnectIosHelloWorld.xcworkspace`
3. `CMD+R`

In other words:

1. Clone.
2. Open TelenorConnectIosHelloWorld.xcworkspace.
3. Build and run.
